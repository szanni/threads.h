threads.h
=========

C11 `<threads.h>` shim for MSVC.

Origins
-------

Originally written as a [gist](https://gist.github.com/yohhoy/2223710).

The source code here consists of that gist and some of the modifications pulled
from the Mesa 3D graphics library.
